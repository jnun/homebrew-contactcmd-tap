# homebrew-contactcmd-tap

Homebrew tap for [ContactCMD](https://contactcmd.com) — a personal CRM for the command line.

## Install

```bash
brew install jnun/contactcmd-tap/contactcmd
```

## Update

```bash
brew update
brew upgrade contactcmd
```

## Uninstall

```bash
brew uninstall contactcmd
brew untap jnun/contactcmd-tap
```
